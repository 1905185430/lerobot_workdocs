## 二、USB 固定端口映射

### 2.1 机械臂串口

通过 `/dev/serial/by-id/` 路径为每台机械臂分配固定标识，解决拔插后 `ttyACM0/1/2/3` 顺序变化的问题：

| 臂名称 | 角色 | by-id 路径 | 当前设备 |
|---|---|---|---|
| so101_cong_left | follower（从臂） | `/dev/serial/by-id/usb-1a86_USB_Single_Serial_5B42073876-if00` | ttyACM3 |
| so101_cong_right | follower（从臂） | `/dev/serial/by-id/usb-1a86_USB_Single_Serial_5B41532613-if00` | ttyACM1 |
| so101_zhu_left | leader（主臂） | `/dev/serial/by-id/usb-1a86_USB_Single_Serial_5B41533034-if00` | ttyACM2 |
| so101_zhu_right | leader（主臂） | `/dev/serial/by-id/usb-1a86_USB_Single_Serial_5B42137834-if00` | ttyACM0 |

### 2.2 相机端口

| 设备 | by-id 路径 | 说明 |
|---|---|---|
| 笔记本前置摄像头 | `/dev/v4l/by-id/usb-Sonix_Technology_Co.__Ltd._USB2.0_HD_UVC_WebCam-video-index0` | → video0 |
| 相机1（副摄） | `/dev/v4l/by-id/usb-Sonix_Technology_Co.__Ltd._USB2.0_HD_UVC_WebCam-video-index1` | → video1 |

> 注意：当前相机识别使用的是笔记本自带摄像头，机械臂相机尚未接入完整测试。相机端口通过 `video-index0/1` 区分，同型号相机共用 Serial 时需用 `by-path` 进一步区分。

### 2.3 相机识别 GUI 工具

开发了图形化相机识别工具 `camera_identifier.py`，功能包括：
- 自动检测所有 `/dev/video*` 设备
- 实时预览画面，确认相机身份
- 手动输入或从预设名称中选择分配
- 保存 `camera_mapping.json`、`camera_mapping.md` 和 `udev` 规则文件
- 预设名称列表可通过 `camera_names.txt` 自定义

## 三、遥操作系统

### 3.1 目录结构

```
SO101_teleoperate/
└── teleoperate.py          # 遥操作主程序

SO101_calibrate/
├── so101_cong_left.py      # 左从臂校准脚本
├── so101_cong_right.py     # 右从臂校准脚本
├── so101_zhu_left.py       # 左主臂校准脚本
├── so101_zhu_right.py      # 右主臂校准脚本
├── USB端口映射表.md         # USB 固定端口文档
├── camera_identifier.py    # 相机识别 GUI 工具
├── camera_names.txt        # 预设相机名称配置
├── camera_mapping.json     # 相机映射结果（运行后生成）
├── camera_mapping.md       # 相机映射 Markdown（运行后生成）
├── 99-so101-cameras.rules  # udev 规则（运行后生成）
└── camera_photos/          # 相机识别时拍摄的预览照片
```

### 3.2 遥操作三种模式

| 模式 | 说明 | 命令 |
|---|---|---|
| 一对一 | 单主臂控制单从臂（左主控左从臂，或右主控右从臂） | `python teleoperate.py` 选模式 1 |
| 一对多 | 单主臂同步控制双从臂（默认右主臂控制左右从臂） | `python teleoperate.py` 选模式 2 |
| 双臂遥操作 | 双主臂独立控制双从臂（左主控左从臂，右主控右从臂） | `python teleoperate.py` 选模式 3 |

## 五、经验记录

1. **USB 固定端口**：机械臂使用 CH340/CH9102F USB 芯片，通过 `by-id` 路径可实现拔插顺序无关的固定识别
2. **同型号相机区分**：icSpring 相机 Serial 相同，换 USB 端口后无法用 `by-id` 区分，需使用 `by-path` 或 udev 规则绑定
3. **波特率**：SO-101 机械臂必须使用 500000 波特率（CH9101F 芯片 1Mbps 不稳定）
4. **PySimpleGUI 版本兼容性**：PySimpleGUI 6.0 不支持 `hover_colors`、`selected_color`、`line_width` 等参数，UI 开发时需注意
