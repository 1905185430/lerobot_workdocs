# SO-101 机械臂

## 概述

SO-101 是 The Robot Studio 设计、Seeed Studio 生产的开源 6 自由度机械臂，使用 Feetech STS 系列舵机。是 LeRobot 生态中官方支持的机器人平台之一。

## 关键参数

- 自由度：6（含夹爪 gripper）
- 舵机：Feetech STS 系列
- 通信协议：SCS/STS 半双工串行
- 默认波特率：1,000,000 bps（LeRobot 默认，实测部分 USB 芯片不支持，需降至 500,000）
- USB 转串口芯片：沁恒 CH9101F（1a86:55d3，cdc_acm 驱动）

## 双臂配置

SO-101 支持主从遥操作配置：
- **从臂 (Follower)**：执行实际动作的机械臂
- **主臂 (Leader)**：人手操控的引导臂
- 支持双臂（bimanual）配置：bi_so_follower/bi_so_leader

## 已知问题

- **[[so101-baudrate-issue|波特率兼容问题]]**：CH9101F 芯片不支持 1Mbps，需修改 LeRobot 默认波特率
- **串口通信偶发丢包**：遥操作脚本需加重试逻辑，disconnect 也需 try/except
- **ttyACM 编号变化**：每次插拔 USB 后端口可能互换

## 配套相机

常与 Orbbec 深度相机配合使用，为 [[vla|VLA]] 模型提供视觉输入。
